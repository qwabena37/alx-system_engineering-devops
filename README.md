# alx-system_engineering-devops

### Task 0. <o>

**File:** `0-alias`

**Description:**  
Creates a shell alias named `ls` that executes `rm *`.

**Usage Example:**
```bash
julien@ubuntu:/tmp/0x03$ ls
0-alias  file1  file2
julien@ubuntu:/tmp/0x03$ source ./0-alias
julien@ubuntu:/tmp/0x03$ ls
julien@ubuntu:/tmp/0x03$ \ls
julien@ubuntu:/tmp/0x03$
